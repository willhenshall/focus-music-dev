File Structure

Each CSV file is named: {channel_name}_{energy_level}.csv

Examples:

  • naturebeat_low.csv
  • naturebeat_medium.csv
  • naturebeat_high.csv
  • the_drop_low.csv
  • chinchilla_medium.csv

CSV Format

Each CSV contains track information with columns like:

  • track_id - Unique identifier for each track
  • bpm - Beats per minute
  • energy - Energy level (0-1)
  • valence - Mood/positivity (0-1)
  • weight - Selection probability weight
